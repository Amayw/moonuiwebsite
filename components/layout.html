# 选项

提供基础布局，样式自定义。

# 组件概述

  • m-layout ：布局容器，其下可嵌套 Header Aside Content Footer 或 Layout 本身，可以放在任何父容器中。

  • m-header ：顶部布局，其下可嵌套任何元素，只能放在 Layout 中。

  • m-aside ：侧边栏，其下可嵌套任何元素，只能放在 Layout 中。

  • m-content ：内容部分，其下可嵌套任何元素，只能放在 Layout 中。

  • m-footer ：底部布局，其下可嵌套任何元素，只能放在 Layout 中。
